
Tagger Tags

Scryfall runs the Tagger project, which allows Scryfall supporters in the community to tag the artwork and functional pieces of each Magic card. We import these tags daily, and you can search for them on Scryfall.

You can use art:, atag:, or arttag: to find things in a card’s illustration, like squirrels. When tagging card artwork, we try to tag what we see, and not what we interpret or what is implied. For example, kindness is not a tag, but things like healing or sunlight are.

You can use function:, otag:, or oracletag: to find “Oracle” tags which describe the function of the card, such as cards that cause removal.

You can edit these tags. Yes, you! Create a Scryfall account and become a Scryfall supporter and then jump into Tagger!
