Thank you for your interest in contributing to OffMeta! Before contributing, please read this document carefully.

1. **Assignment of Rights**: You hereby assign all intellectual property rights in your contribution to OffMeta. This includes copyright, patent rights, and any other proprietary rights.

3. **No Compensation**: Contributions are made voluntarily without expectation of compensation unless otherwise agreed in writing.

4. **Confidentiality**: You agree to keep any non-public information about the project confidential.

## How to Contribute

If you'd like to contribute, please:

1. **Contact OffMeta first** - Reach out to discuss your proposed contribution before starting work
2. **Sign a Contributor License Agreement (CLA)** - You may be required to sign a formal CLA before your contribution can be accepted
3. **Submit via Pull Request** - Once approved, submit your contribution as a pull request

## Code Standards

- Follow existing code style and conventions
- Write clear, descriptive commit messages
- Include tests where applicable
- Update documentation as needed

## Questions?

For licensing or contribution inquiries, please contact OffMeta.
