# Homepage Activation Plan

## Above-the-fold hierarchy

1. Hero message clarifying that OffMeta is a plain-English MTG card search tool.
2. Search bar with support copy, trust signals, and 6–8 high-intent example chips.
3. Micro-demo showing natural language → translated Scryfall query → result preview.
4. Curated searches as the first post-hero discovery module.
5. Recent searches next for returning users.
6. FAQ lower on the page so it supports evaluation instead of competing with activation.

## Hero copy shipped

- **Primary headline:** Search Magic cards in plain English.
- **Primary subheadline:** OffMeta is for Commander players, brewers, and deckbuilders who want to search MTG cards in plain English and get instant translated results without learning Scryfall syntax.
- **Support copy:** Type what you want, like “budget board wipes under $5” or “cards that protect my commander,” and OffMeta turns it into a real card search instantly.

## A/B test variants prepared

- **Variant A:** Search Magic cards in plain English.
- **Variant B:** Find MTG cards without learning Scryfall syntax.
- **Variant C:** Describe the card you want — OffMeta finds it instantly.

## Example query set

Priority order:

1. find budget board wipes under $5
2. cards that protect my commander
3. cheap graveyard hate for EDH
4. mana rocks that cost 2
5. best black removal for commander
6. cards that double ETB triggers
7. one-card combos in Simic colors
8. cheap white protection spells

## Why this ordering

- Starts with common Commander/deckbuilding jobs users can validate instantly.
- Mixes easy-win utility searches with “wow” capability searches.
- Keeps the primary activation path focused on typing or tapping a chip first.
