# Internationalization

This page is the short index for the locale system.

## Supported locales

- `en`, `es`, `fr`, `de`, `it`, `pt`, `ja`, `ko`, `ru`, `zhs`, `zht`

## Canonical files

- `src/lib/i18n/index.tsx`
- `src/lib/i18n/constants.ts`
- `src/lib/i18n/*.json`

## Key behavior

- English is the fallback source of truth
- Locale is persisted in localStorage
- Search requests can carry locale into semantic-search
- Scryfall uses locale mapping for localized card results

## More detail

For implementation notes and key conventions, see the source files above and the app-level translation behavior in `src/components/` and `src/hooks/`.
